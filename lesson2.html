レッスン 2

変数と基本データ型

学習時間: 約15分

1. Pythonの変数

変数は、データを格納するためのメモリ上の「名前付き箱」のようなものです。Pythonでは、変数の宣言方法はとてもシンプルです。

変数の基本

# 変数の宣言と値の代入
name = "Python"
age = 30
is_programming = True

# 値の出力
print(name)  # 出力: Python
print(age)   # 出力: 30
print(is_programming)  # 出力: True

Pythonの変数命名規則

  • アルファベット、数字、アンダースコア(_)が使用可能
  • 数字から始めることはできない
  • 大文字と小文字は区別される (name と Name は別の変数)
  • 予約語（if, for, whileなど）は使用できない

2. 基本データ型

Pythonには、以下のような基本的なデータ型があります。

数値型

# 整数型 (int)
age = 30
count = -5

# 浮動小数点型 (float)
height = 175.6
temperature = -2.5

# 基本的な演算
sum_value = 10 + 5  # 15
difference = 10 - 5  # 5
product = 10 * 5  # 50
division = 10 / 5  # 2.0 (浮動小数点数として返される)
integer_division = 10 // 3  # 3 (小数点以下切り捨て)
remainder = 10 % 3  # 1 (余り)
power = 2 ** 3  # 8 (2の3乗)

文字列型 (str)

# 文字列の宣言
name = "Python"
message = 'Hello, World!'

# 複数行の文字列
multi_line = """これは
複数行の
文字列です。"""

# 文字列の連結
greeting = "Hello, " + name  # "Hello, Python"

# 文字列の繰り返し
repeated = "Python" * 3  # "PythonPythonPython"

# 文字列の長さ
length = len(name)  # 6

# 文字列のインデックス（位置）
first_char = name[0]  # "P"
last_char = name[-1]  # "n"

# 文字列のスライス（一部を取得）
substring = name[1:4]  # "yth"

ブール型 (bool)

# ブール値（真偽値）
is_active = True
is_completed = False

# 比較演算子
is_equal = (10 == 10)  # True
is_not_equal = (10 != 5)  # True
is_greater = (10 > 5)  # True
is_less = (10 < 20)  # True
is_greater_equal = (10 >= 10)  # True
is_less_equal = (5 <= 10)  # True

# 論理演算子
result1 = True and False  # False
result2 = True or False  # True
result3 = not True  # False

3. データ型の確認と変換

変数のデータ型を確認したり、別の型に変換したりする方法を見てみましょう。

型の確認

# type()関数でデータ型を確認
x = 10
y = "Hello"
z = 3.14

print(type(x))  # 
print(type(y))  # 
print(type(z))  #

型の変換

# 整数から文字列への変換
age = 30
age_str = str(age)  # "30"

# 文字列から整数への変換
num_str = "42"
num = int(num_str)  # 42

# 整数から浮動小数点数への変換
value = 5
value_float = float(value)  # 5.0

# 浮動小数点数から整数への変換（小数点以下は切り捨て）
pi = 3.14159
pi_int = int(pi)  # 3

4. 入力の受け取り

ユーザーからの入力を受け取るには、input()関数を使用します。

# ユーザーからの入力を受け取る
name = input("あなたの名前を入力してください: ")
print("こんにちは、" + name + "さん！")

# 数値の入力（文字列として受け取られるため、変換が必要）
age_str = input("あなたの年齢を入力してください: ")
age = int(age_str)  # 文字列を整数に変換
print("あなたは", age, "歳です。")

実践課題

今日のレッスンで学んだ内容を確認するための課題です。15分以内で完了できるはずです。

  1. 自分の名前、年齢、趣味を変数に格納し、それぞれを出力するプログラムを作成してください。
  2. 円の面積を計算するプログラムを作成してください。半径はユーザーから入力を受け取ります。（面積 = πr²、πは3.14とします）
  3. 摂氏から華氏への温度変換プログラムを作成してください。（華氏 = 摂氏 × 9/5 + 32）
  4. 任意の文字列の長さを数え、その文字列の最初と最後の文字を出力するプログラムを作成してください。
