レッスン 1

Pythonの基本と環境構築

学習時間: 約15分

1. Pythonとは？

Pythonは、シンプルで読みやすい構文を持つ高水準プログラミング言語です。データ分析、ウェブ開発、人工知能など、幅広い分野で使用されています。

Pythonの特徴

  • 読みやすく、書きやすい構文
  • 幅広いライブラリとフレームワーク
  • 多様なプラットフォームで動作
  • プロトタイピングに適した開発速度

2. 環境構築

Pythonを使用するには、まずお使いのコンピュータに環境を整える必要があります。

Pythonのインストール

  1. Python公式サイトからインストーラーをダウンロード
  2. インストーラーを実行（「Add Python to PATH」にチェックを入れることを忘れないでください）
  3. インストールが完了したら、コマンドプロンプトまたはターミナルで動作確認

インストール確認

python --version

上記コマンドを実行すると、インストールされたPythonのバージョンが表示されます。

3. 最初のPythonプログラム

Pythonの基本を理解するため、簡単な「Hello, World!」プログラムを作成してみましょう。

Hello, World!

print("Hello, World!")

このコードをファイル（例: hello.py）に保存し、ターミナルで実行します。

python hello.py

出力: Hello, World!

4. 統合開発環境 (IDE)

プログラミングをより効率的に行うために、IDEの使用をお勧めします。人気のある選択肢には以下があります。

Visual Studio Code

Microsoft社が開発した無料のコードエディタで、Python拡張機能を追加することで強力なIDEになります。

ダウンロード

PyCharm

JetBrains社によるPython専用の統合開発環境。コミュニティ版は無料で使用できます。
